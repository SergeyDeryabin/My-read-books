# Выдержка из книги:


## Предисловие

Книга «Разработка с фреймворком jQuery 2.0. Книга рецептов» предоставит вам много рецептов для повторно используемого кода, чтобы создать общий и уникальный веб-сайт и элементы веб-приложения, плагины и интерфейсы, используя самую популярный клиентский фреймворк jQuery. Следование пошаговым инструкциям для каждого из рецептов не только предоставит вам применимый код, но также и понимание необходимости его расширения и изменить к лучшему.

 


## Для кого предназначена эта книга?

 Эта книга для любого, кто или новичок в использовании фреймворка jQuery и надеющийся изучать некоторые его основы, или знаток в использовании фреймворка jQuery и надеющийся расширить свое знание и создавать некоторые усовершенствованные компоненты для их веб-сайта или веб-приложения.  Эта книга - превосходный ресурс для веб-разработчиков всех уровней знаний и уровней навыка. 
  
 

## Что рассматривает эта книга?
 
**Глава 1 «Манипулирование объектной моделью документа»**  рассматривает то, как использовать фреймворк jQuery, чтобы манипулировать HTML-кодом в объёме 186 веб-страниц, на клиенте, чтобы создать богатое и визуальное впечатление для пользователя.

**Глава 2 «Взаимодействие с пользователем путем использования jQuery-событий»** использует мощь фреймворка jQuery для обнаружения и ответа на взаимодействие с пользователем, что создает интуитивные пользовательские интерфейсы.

**Глава 3 «Загрузка и манипулирование динамическим содержанием с помощью AJAX и JSON»**  использует AJAX-функциональность фреймворка jQuery с JSON-отформатированными данными, чтобы реализовать обновление содержания на странице, без необходимости в обновлении самой страницы.

**Глава 4 «Добавление привлекательных видов из модуля эффектов jQuery Effects»** объясняет, как добавить сияние к веб-сайту или веб-приложению с помощью  эффектов класса jQuery Effects и базовыми анимациями, для создания незабываемых проектов.

**Глава 5 «Обработка формы»** описывает, как использовать фреймворк jQuery, для построения на стороне клиента устойчивой проверки допустимости  и интуитивного впечатления пользователя для веб-форм.

**Глава 6 «UI-интерфейс пользователя»** подробно объясняет, как переделать форму и с нуля создать мощно интуитивные интерфейсы и привлечь пользователя с помощью высокого уровня интерактивности.

**Глава 7 «Анимация UI-интерфейса пользователя»**  подробно объясняет, как расширить встроенную  jQuery-анимацию и комбинировать CSS-таблицу стилей с фреймворком jQuery, чтобы создать невероятные модули веб-сайта, для использования с любым веб-сайтом.

**Глава 8 «Понимание разработки плагинов»**  объясняет, как создать повторно используемый код, предоставляющий решения широкого диапазона общих проблем веб-сайтов и веб-приложений.

**Глава 9 «Библиотека виджетов jQuery UI»**   рассматривает, как с помощью  jQuery UI-библиотеки виджетов интерфейса пользователя улучшить веб-сайт или веб-приложение путем создания привлекательных и удобных для пользователя элементов страниц и интерфейсов.

**Глава 10 «Работа с фреймворком jQuery Mobile»**  объясняет, как создать мобильный и кросс-платформенно-готовый веб-сайт, используя мощный фреймворк для мобильных устройств Mobile, на базе фреймворка jQuery.

 
 
## Что потребуется для работы с книгой?

Для всех рецептов в этой книге требуется интегрированная IDE-среда разработки, для написания JavaScript-кода, HTML-кода и CSS-кода, и требуется веб-браузер, для выполнения написанного кода. Для некоторых более усовершенствованных рецептов в этой книге требуется веб-сервер, выполняющий СУБД MySQL и интерпретатор PHP-скриптов.
